# LocalMusicPlayer
本地音乐播放器demo</br>
功能描述：</br>
1、遍历本地音乐文件</br>
2、音乐播放暂停</br>
3、播放进度控制</br>
4、音乐的搜索</br>
5、通知栏控制音乐的播放</br>
